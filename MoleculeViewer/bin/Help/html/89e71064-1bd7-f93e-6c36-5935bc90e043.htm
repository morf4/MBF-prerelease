The current molecule coloring method.

CreateAtomsAndStructures(Stream)
Called by the contructor to parse the portions of the PDB file related to atoms and secondary structures.

CreateBackbone()()
Called by the contructor after creating the Atom objects to identify the backbone atoms and connect them via referneces.

CreateBonds()()
Called by the contructor after creating the Atom objects to identify covalently bonded atoms. Uses a simple distance heuristic of six angstroms.

CreateChains()()
Called by the constructor to create Chain objects and group their constituent residues (amino acids).

CreateModel()()
Called by the constructor to create the container Model3DGroup for the molecule's 3D model.

CreateMoleculeTransform()()
Called by the constructor to calculate the default scale and translation of the molecule based on the bounding box of the atoms.

CreateResidues()()
Called by the constructor to create Residue objects and group their constituent atoms.

CreateRibbons()()
Called by the constructor to create Ribbon objects which are used to compute the spline curves for secondary struction representations.

GetSelectionViewState()()
Calculates a decent orientation of the molecule based on the bounding box of the currently selected atoms. Used for animating to the center of a selection.

Model
The Model3D for the molecule which contains all constituent parts. This can be added to a Visual3D and then diplayed in a Viewport3D.

MoleculeTransform
The current orientation of the molecule. This is used by a StructureControl

Residues
All of the residues (amino acids) contained in the molecule.

SelectedAtoms
Provides a list of all currently selected atoms.

SetStructureInfo()()
Called by the constructor to set secondary structure related properties on the residues and atoms that compose each structure.

Remarks
This class contains references to the other business objects that make up a molecule and centralizes the functionality to generate the necessary 3D meshes.
